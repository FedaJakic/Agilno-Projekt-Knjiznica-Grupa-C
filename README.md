# Agilno vođenje projekta ~ Grupa C ~
## Knjižnica Marko Marulić

Tehnologije:
- ReactJS
- NodeJS
- PostreSQL

## Paketi

#### Root

```http
  npm i -D nodemon concurrently
  npm i dotenv
  npm i express-async-handler
```

#### Fronted
```http
  npm install
  npm i react-bootstrap
  npm i axios
```

## Pokretanje
### Cijeli projekt
```http
  npm run dev
```

### Samo Frontend
```http
  npm run client
```

### Samo Backend
```http
  npm run server
```
